Allows you to use InvoiceBox payment gateway with the WooCommerce 3 plugin.

== Description ==

Internet shops: connect payment systems: WORLD, Visa, MasterCard, Alipay, Internet-Banks, ATMs, E- money, as well as payments from customers abroad. El. Checks for 54-FZ for you!

For work it is necessary to conclude an agreement InvoiceBox [InvoiceBox](https://www.invoicebox.ru/ru/ "InvoiceBox").

= Key features =

* Commission from 2 to 3.3% when paying with bank cards.
* Does not require an online box office.
* Checks for 54-FZ.
* Support WooCommerce 3.


== Installation ==

1. Install plugin from WordPress Plugin Directory or upload `invoicebox-for-woocommerce` to the `/wp-content/plugins/` directory manually.
2. Activate plugin through the 'Plugins' menu in WordPress admin panel.
